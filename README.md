# Weather App ☀️🌧️<br>
A sleek and user-friendly Weather App built with Flutter that allows users to explore real-time weather data for over 100+ cities. This app is perfect for those who want to keep track of weather conditions at any location of their interest!<br>

Features ✨<br>
🌍 100+ Cities Support: Browse and check the weather in over 100 cities worldwide.<br>
🌦️ Real-Time Weather Data: Provides up-to-date weather information, including temperature, humidity, wind speed, and weather conditions.<br>
📍 City Search: Easily search for cities by name to get weather updates.<br>
🌐 Third-Party Weather API Integration: Fetches live weather data dynamically using a third-party API.<br>


Tech Stack 🛠️<br>
Flutter: Cross-platform development framework.<br>
Dart: Programming language used with Flutter.<br>
Third-Party Weather API: Used to fetch real-time weather data.rom open weather api <br>
